**Online Bookstore Sales Dashboard | Power BI :**

Developed an interactive Power BI dashboard to analyze bookstore sales data across multiple years (2022–2024). The dashboard provides insights into revenue by genre, monthly sales trends, order quantities, country-wise performance, and top-selling authors. Implemented data modeling, visual analytics, and interactive slicers to support business intelligence and decision-making.

Tools Used: Power BI, DAX, Data Modeling, Data Visualization

Key Features:

Year-wise sales analysis (2022–2024)
Genre-wise revenue and order quantity insights
Monthly sales trend analysis
Country-wise order distribution
Top-performing authors and sales tracking
Interactive filters and dynamic visualizations
